## About
Wonder packages 40 decorative motifs as original PNG files. Each token uses the `wonder.*` prefix. Shapes are preserved exactly as exported — form, color, and grain are not modified.

## Motifs
- Cross (4 colorways)
- Dot grid (4 colorways)
- Radial rays (4 colorways)
- Loopy scribble (4 colorways)
- S-curve link (4 colorways)
- Cloud (2 colorways)
- Wavy line (4 colorways)
- Concave star (1 colorway)
- Thick arch (4 colorways)
- Half circle (4 colorways)
- Sun burst (1 colorway)
- Circle (4 colorways)

## Assets
All files live in `Wonder/assets/` as `wonder-{motif}-{color}.png`.
