01

Overview

Fairshare is a collaborative platform that makes managing a shared household easier and more enjoyable. It addresses the everyday challenges of living together by centralizing task management, helping everyone stay on the same page and remain accountable.

Instead of boring checklists, Fairshare features a fun point system that motivates everyone to develop good habits and turn routine chores into shared successes. The platform effectively integrates everyone's schedules so that every resident can communicate through a single, unified channel.

02

Challenge

The challenge for Fairshare was designing a system that balances individual autonomy with collective transparency. Users have unique schedules and preferences. At the same time, the household needs a reliable source of truth to track responsibilities and ensure accountability.

The goal was to create a lightweight, user-friendly system that supports daily use, yet is robust enough to maintain accurate synchronization across multiple users. This solution enables households to function smoothly while respecting individual needs.

06

Visual Style Deployment

I curated a mood board to define the aesthetic direction, prioritizing a clean, functional interface. I formalized these design system specifications in a DESIGN.md file within Cursor, ensuring consistency throughout the build. The color palette draws inspiration from our team's working boards, using lively Post-it colors to evoke collaboration and creativity, making the platform feel more like a shared workspace.
