## Overview
Fairshare is a collaborative platform that makes managing a shared household easier and more enjoyable. It addresses the everyday challenges of living together by centralizing task management, helping everyone stay on the same page and remain accountable. 

Instead of boring checklists, Fairshare features a fun point system that motivates everyone to develop good habits and turn routine chores into shared successes. The platform effectively integrates everyone's schedules so that every resident can communicate through a single, unified channel.

## Challenge
The challenge for Fairshare was designing a system that balances individual autonomy with collective transparency. Users have unique schedules and preferences. At the same time, the household needs a reliable source of truth to track responsibilities and ensure accountability. 

The goal was to create a lightweight, user-friendly system that supports daily use, yet is robust enough to maintain accurate synchronization across multiple users. This solution enables households to function smoothly while respecting individual needs.

## Ideation
Before starting the design process, I outlined possible scenarios and user paths to gain a clearer understanding of how users would interact. By doing this helped me creating comprehensive user flows and a sitemap to ensure the navigation felt logical for diverse household types.

### User Flow
By creating user flows based on these scenarios, I was able to visualize the different paths and screens involved in the user journey. This exercise was instrumental in identifying key interactions and ensuring a seamless experience.
flow: Assets/01-Flow-Fairshare.png

### Site Map
Once I had a better grasp of the user flows, I developed a sitemap to organize the structure and content of the application. The sitemap served as a guide for the overall architecture, helping me ensure that every necessary pathway and screen was accounted for.

sitemap: Assets/01-Sitemap.png

After looking at various household management apps, I noticed some key areas where they fall short, especially in task delegation and gamification. This inspired my approach to a point system for my app.
